# GitHub Copilot（学生向け）

GitHub Copilot は AI を活用したコーディング支援ツールです。学生の方は、**GitHub Student Developer Pack** を通じて無料で利用できる場合があります。

## 1. GitHub Student Developer Pack への申請

1. [https://education.github.com/pack](https://education.github.com/pack) にアクセスします。
2. **Sign up for Student Developer Pack** をクリックします。
3. GitHub アカウントでサインインします。
4. 大学のメールアドレスまたは学生証を使って学生であることを確認します。
5. 審査完了を待ちます（通常数日以内）。

## 2. GitHub Copilot の有効化

学生申請が承認されたら：

1. [https://github.com/settings/copilot](https://github.com/settings/copilot) にアクセスします。
2. Copilot をアカウントで有効にします。

## 3. VS Code に Copilot 拡張機能をインストール

1. VS Code を開きます。
2. **拡張機能**パネルを開きます（Ctrl+Shift+X / Cmd+Shift+X）。
3. **GitHub Copilot** を検索します。
4. **インストール**をクリックします。
5. プロンプトに従って GitHub にサインインします。

## 4. Copilot の使い方

- コードを入力すると Copilot が候補を提案します。**Tab** キーで候補を確定します。
- 候補を却下するには **Escape** キーを押します。
- **Ctrl+Enter**（Mac では **Cmd+Enter**）で複数の候補パネルを開きます。

## 5. Copilot Chat（任意）

利用可能な場合は、**GitHub Copilot Chat** 拡張機能をインストールすることで、VS Code 内でインタラクティブな AI アシスタントを使用できます。

*注意：利用資格や機能は変更される場合があります。最新情報は [GitHub Education](https://education.github.com/) ページをご確認ください。*

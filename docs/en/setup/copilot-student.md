# GitHub Copilot (Student)

GitHub Copilot is an AI-powered coding assistant. As a student, you may be eligible for free access through the **GitHub Student Developer Pack**.

## 1. Apply for the GitHub Student Developer Pack

1. Go to [https://education.github.com/pack](https://education.github.com/pack)
2. Click **Sign up for Student Developer Pack**.
3. Sign in with your GitHub account.
4. Verify your student status using your institutional email or student ID.
5. Wait for approval (usually within a few days).

## 2. Enable GitHub Copilot

Once your student application is approved:

1. Go to [https://github.com/settings/copilot](https://github.com/settings/copilot)
2. Enable Copilot for your account.

## 3. Install the Copilot Extension in VS Code

1. Open VS Code.
2. Go to the **Extensions** panel (Ctrl+Shift+X / Cmd+Shift+X).
3. Search for **GitHub Copilot**.
4. Click **Install**.
5. Sign in to GitHub when prompted.

## 4. Using Copilot

- Copilot suggests code as you type — press **Tab** to accept a suggestion.
- Press **Escape** to dismiss a suggestion.
- Use **Ctrl+Enter** (or **Cmd+Enter** on Mac) to open a panel with multiple suggestions.

## 5. Copilot Chat (Optional)

If available, install the **GitHub Copilot Chat** extension for an interactive AI assistant inside VS Code.

---

*Note: Eligibility and features may change. Check the [GitHub Education](https://education.github.com/) page for the latest information.*
